CIDR란 기존의 클래스 체계보다 유연하고 효율적으로 IP 주소를 사용할 수 있도록 하는 기법입니다.
기준 아이피/서브넷 마스크로 표현합니다. 예를 들어 192.168.32.1/24라는 주소의 기준 아이피는 192.168.32.1 이고, 24는 서브넷 마스크를 의미합니다.
